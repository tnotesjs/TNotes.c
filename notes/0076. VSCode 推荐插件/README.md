- 学习 C/C++ 开发推荐使用的 VSCode 插件

## 2. 评价

这篇笔记用于记录学习 C/C++ 开发推荐使用的 VSCode 插件列表。

以下是将所选插件列表整理后的 Markdown 表格：

| 插件 ID | 插件名称 | 说明 |
| --- | --- | --- |
| `ms-vscode.cpptools` | C/C++ | 核心插件，提供 IntelliSense、调试、代码浏览 |
| `ms-vscode.cpptools-extension-pack` | C/C++ Extension Pack | 包含 C/C++、CMake Tools 等插件的合集 |
| `ms-vscode.cpptools-themes` | C/C++ Themes | 为 C/C++ 提供增强的语法高亮主题 |
| `ms-vscode.cmake-tools` | CMake Tools | CMake 项目构建、配置、调试支持 |
| `twxs.cmake` | CMake | CMake 语法高亮和代码片段 |
| `formulahendry.code-runner` | Code Runner | 一键运行 C/C++ 代码，适合快速验证小程序 |
| `hediet.debug-visualizer` | Debug Visualizer | 调试时可视化数据结构（数组、链表等） |
| `usernamehw.errorlens` | Error Lens | 内联显示编译错误和警告，便于快速定位问题 |
| `aaron-bond.better-comments` | Better Comments | 高亮注释分类（TODO、警告等），提升代码可读性 |
| `vadimcn.vscode-lldb` | CodeLLDB | 基于 LLDB 的调试器，macOS 上调试 C/C++ 的首选 |
| `jeff-hykin.better-cpp-syntax` | Better C++ Syntax | 增强 C++ 语法高亮，支持更多现代 C++ 特性 |
| `cschlosser.doxdocgen` | Doxygen Documentation Generator | 自动生成 Doxygen 风格注释，培养文档习惯 |
| `ms-vscode.makefile-tools` | Makefile Tools | Makefile 项目的智能感知和构建支持 |

可以提供插件 ID 在【扩展：商店】中精准定位到具体的插件，比如搜 `ms-vscode.cpptools`：
